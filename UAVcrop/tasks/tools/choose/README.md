# 抽样说明
# 设置参数：
    原图大小                            例：img_size = (2048, 2048)
    切图大小                            例：cut_size = (1024, 1024)
    原图文件地址                         import_path = ''
    单张原图代表的实际面积                 area = 1
    置信度                              例：confidence = 0.95
    抽样数量                             默认：n = 30
    期望偏差值百分比                      例：expected_percent = 0.05
    切图后保存地址                        cut_path = ''
    预测图片（带框）保存地址                output_path = ''
    单次预测图片数量                      predict_count = 10
    允许的图片类型                        allowed_types = {'jpg', 'png', 'jpeg'}
    是否将被抽样的切图合成原图并保存         if_return_img
    合成图片保存地址                      final_result_img = ''
# 业务逻辑：
    1. 判断原图文件数量，若小于抽样数量n返回空tuple()
    2. 从原图文件夹中抽取n个图片
    3. 将n个原图切图，将切割的小图保存在切图文件夹中
    4. 从切图文件夹中选取n个小图进行批量的图片预测，并将预测结果图片保存，上写苗数、苗/亩（或其他面积单位）数
    5. 根据预测结果、抽样数量、样本总数等，计算置信区间值、偏差的百分比，并判断是否符合预期
        （1）符合预期：将结果返回
        （2）不符合预期：
                a. 计算应该抽取的样本数m
                b. 若没有被切图的剩余大图数量大于等于（m-n），抽取（m-n）个大图；反之抽取剩余全部大图
                c. 将新抽取的大图进行切图，混入原有切图的集合
                d. 从混合后的切图集合中，获取没有被抽样过的切图集合，补充抽取（m-n）个切图进行批量预测
                e. 根据预测结果计算置信区间值、偏差的百分比并返回
    6. 如果需要合成结果图片：
        （1）从字典中获取被抽样过的图片名的set集合
        （2）遍历集合，创建原图大小的空图片，用于粘贴对应位置的小图
        （3）获取小图：
                a. 如果图片进行过预测，则取预测过的（画框过的）图片
                b. 图片没有被预测过，取切图结果文件夹中对应的小图
        （4）将小图粘贴在大图对应位置，保存生成最终的结果图
