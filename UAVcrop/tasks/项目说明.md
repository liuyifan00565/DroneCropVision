# 说明
tasks目录下的代码是北大荒信息有限公司的前期代码，可以参考

# 项目简介
本项目基于[ultralytics](https://github.com/ultralytics/ultralytics)
改造，主要处理目标检测类问题。<br>
开发人员应根据不同问题在tasks文件夹中创建子文件夹，用于保存问题相关的代码、模型和文档，使得项目定制功能与源项目可以独立演进。<br>
当需要升级ultralytics时，直接覆盖本项目对应文件即可。<br>
注意：当需要修改ultralytics源代码时，请在本文档中说明，以便升级ultralytics后进行相应调整。

# 目录结构

    AI计数,拍照数种子数量
