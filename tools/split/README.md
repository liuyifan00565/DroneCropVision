# tif_split.py说明
# 需要参数：
    1. 带有经纬度信息的tif文件        url = ''
    2. 图片中有效的实际面积           area_total = 1 
    3. 黄色警告值                   yellow_warn = 0.9
    4. 红色警告值                   red_warn = 0.5
    5. tif切图的小图保存路径          out_put_path = ''
    6. 预测结果图片的保存路径          predict_output_path = ''
    7. 单次预测的图片数量             predict_count = 10
    8. 预测允许的图片类型             allowed_types = {'jpg', 'png', 'jpeg'}
# 流程：
    1. 确定参数
        （1）tif切图jpg文件以及保存图片信息的txt文件的输出路径
        （2）透明黄色、红色警告图层的占比
        （3）图片预测结果保存路径
        （4）单次预测图片个数
        （5）允许的图片类型
    2. 对tif文件进行处理
        （1）分离tif图层，rbg保存为jpg文件，并获取每张小图的有效像素值
        （2）根据图层4是否有值，计算总体的有效像素值
        （3）将每张小图的经纬度信息、像素信息保存在txt中，以防丢失
        （4）返回结果：整个大图的宽度、高度、横向切图数、纵向切图数、总像素数
    3. 图片预测
        （1）打开txt文件，获取内容
        （2）打开tif切图为jpg的文件夹，获取需要进行预测的图片
        （3）根据设置的单次预测图片数量，进行批量识别
        （4）将识别结果（苗数、标框的结果图片）根据txt中对应图片的像素数与总像素数计算比例，得到单张小图的苗/亩数，并将结果再次存入txt
        （5）累计获得总的苗数，返回结果
    4. 计算得到总的苗/亩数
    5. 图片处理（加入透明图层）
        （1）将预测后的结果图片，与txt中的信息进行对应，得到每张小图的苗/亩数
        （2）将每张小图的苗/亩数，与总的苗/亩数乘以对应黄色、红色警告值的结果进行对比
        （3）小图苗/亩数小于等于红色警告值时，将图片覆盖透明度为50%的红色图层，另存为png
        （4）小图苗/亩数在黄色与红色之间，将图片覆盖透明度为50%的黄色图层，另存为png
    6. 图片拼接
        （1）根据原始tif的宽高信息，创建一张最终结果图
        （2）根据横向切图数、纵向切图数进行遍历
        （3）根据文件名中的i、j获取对应位置的图片，粘贴进最终结果图
        （4）保存最终结果图
